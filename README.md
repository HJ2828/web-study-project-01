# Web_studyProject01
html, css, js를 이용한 웹 제작

## 제작 기간
- 2023.08.24 ~ 제작 중

## 기능
- 사용자 이름 입력 하여 띄우기
- 년, 월, 일, 요일, 시간 띄우기
- 시간에 따른 인삿말 변경(morning, afternoon, evening, night)
- 날씨
- 랜덤 배경 변경, 배경 이미지 주소를 입력 통한 배경 변경
- 체크 리스트

## 기술/스택
- html, css, js
- OpenWeatherApi
